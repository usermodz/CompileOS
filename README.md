# CompileOS
A Debian-based Linux distribution focused on coding.

Compile OS is a Debian-based Linux distribution engineered for maximum productivity. It combines the rock-solid stability of Debian with a modern, customized GNOME experience, pre-loaded with the essential 
toolbelt for any modern coding workflow.

## Visual
A refined, professional gray aesthetic optimized for long coding sessions. Made to be clean and simple, with no distractions.

## Toolbelt
Forget about long setup scripts, Compile OS already did most of the work for you.

### Programming Languages:
 - Python
 - Java
 - Rust
 - Go
 - Node.js (LTS)

### Core Tools:
 - Docker and Docker Compose (pre-configured)
 - Git
 - VSCodium

### Modern CLI:
Enhanced terminal experience with:
 - eza (ls)
 - bat (cat)
 - fzf
 - zoxide

### Compilers:
- Full build-essential suite
- Clang
- Ninja Build

## Performance & Stability

### Debian Stable Core:
Built on the world's most stable foundation, updated for 2026 hardware.

### Hybrid Package Management:
Native support for APT, Flatpak (Flathub), and Snap integrated into the GNOME Software Store.

### Zero Bloat:
Every pre-installed application serves a specific purpose in a developer's lifecycle.

## Getting Started
Compile OS uses the Calamares installer, providing a smooth and secure graphical installation.

- Grab the latest ISO from the [Releases] section.
- Use Ventoy or BalenaEtcher to create a bootable USB.
- Enter Live Mode, explore the environment, and click Install Compile OS.
